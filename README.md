This repository contains three beginner-friendly frontend mini projects built using HTML, CSS, and JavaScript.
The projects focus on practicing logic building, user interaction, and responsive web design.

Mini Project 1: Budget Checking App

A simple JavaScript-based application that checks whether a user can afford a product based on their budget.

Features

- Takes user budget and product price

- Uses if / else conditions

- Displays affordability result

- Beginner-friendly logic

Technologies Used

- HTML5
- CSS3

- JavaScript (basic)
  Mini Project 2: Are You Human?

A small logic-based project that determines whether the user is human based on input conditions.
Features

- User input handling

- Conditional logic using if / else

- Simple output display

- Helps practice JavaScript decision making
  Technologies Used

- HTML5
- CSS3
- JavaScript (basic)

Mini Project 3: Personal Portfolio Website

A responsive personal portfolio website created to showcase my profile, skills, projects, and contact information.
Features

- About Me section

- Skills list

- Work / Projects section

- Services section

- Contact

- CV download button

- Responsive design for mobile & tablet

Technologies Used

- HTML5

- CSS3

- Basic responsive design (Flexbox & Media Queries)
